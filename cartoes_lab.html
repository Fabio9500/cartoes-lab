💾 Backup dos dados

Salve uma cópia dos seus dados — localmente, no GitHub ou nos dois ao mesmo tempo.

💾 Backup manual em disco local — passo a passo

1
Clique em "Baixar backup.json" acima — o arquivo será baixado automaticamente para a pasta Downloads do seu computador.
2
Mova o arquivo para uma pasta segura — sugerimos criar Documentos → Cartoes → Backups e renomear com a data: backup-cartoes-24-06-2026.json.
3
Repita ao menos 1 vez por semana ou sempre que lançar muitas compras de uma vez.
4
Para restaurar: clique em "Importar backup" (seção abaixo) e selecione o arquivo JSON salvo. Os dados serão restaurados e sincronizados com o GitHub automaticamente.
💡 Backup extra na nuvem: seus dados também ficam salvos automaticamente no GitHub a cada alteração. Para baixar direto de lá, acesse github.com/Fabio9500/Cartoes, clique no arquivo dados.json (ou dados-pj.json) → botão "Raw" → Ctrl+S para salvar.

📥 Importar backup

Restaura a partir de um arquivo .json exportado anteriormente.

🏦 Importar extrato do cartão

Cole o extrato do cartão (copiado do internet banking) ou envie um arquivo CSV exportado. O sistema identifica as compras automaticamente e evita duplicar lançamentos já existentes.

🔗 Integração com TesourariaSys

Ao marcar uma fatura como paga, você pode debitar automaticamente uma conta do TesourariaSys — sem precisar registrar de novo lá. Precisa de um token do GitHub com permissão de escrita no repositório Fabio9500/tesourariasys.

⚠️ Conflito de Sincronização

Você fez alterações offline neste dispositivo que ainda não foram enviadas à nuvem.

Enquanto isso, outro dispositivo já salvou alterações diferentes no sistema.

Escolha como deseja prosseguir:

Dica: se não tiver certeza, escolha "Manter dados da nuvem" — depois confira o que está salvo e refaça manualmente o que faltar.
