# cartoes-lab

Repositório de **experimentação e aprendizado** com o Claude Code.

## O que é isto

Uma cópia isolada do CartõesPF (sistema real em `Fabio9500/Cartoes`), usada **só para praticar** o fluxo de trabalho com o Claude Code — instalar, clonar, pedir alterações, revisar diffs, dar commit.

- **Não afeta o sistema em produção.** O arquivo `cartoes_lab.html` foi ajustado para gravar seus dados neste repositório (`cartoes-lab/dados.json`), não no repositório real (`Cartoes/dados.json`).
- Os dados aqui são de teste. Pode mexer à vontade, quebrar, testar — é exatamente para isso que existe.

## Como usar

1. Clone este repositório na sua máquina.
2. Rode `claude` dentro da pasta.
3. Peça alterações pequenas pra começar (ex: mudar uma cor, adicionar um texto).
4. Veja o resultado publicando no GitHub Pages: `https://fabio9500.github.io/cartoes-lab/cartoes_lab.html`

## Regras de desenvolvimento

Ver `CLAUDE.md` — o Claude Code lê esse arquivo automaticamente ao iniciar.
